# Java-POO
Estudos de programação orientada a objetos com Java.

**POO (Programação Orientada a Objetos)** é um paradigma de programação que organiza o código em **objetos**, representando elementos do mundo real.

Um objeto possui:

* **Atributos** → características
* **Métodos** → ações

Exemplo:

Uma pessoa possui:

* Nome
* Idade

E pode realizar ações como:

* Falar
* Andar
* Estudar

Em Java:

```java
public class Pessoa {

    String nome;
    int idade;

    public void falar() {
        System.out.println("Olá!");
    }

}
```

A POO tem como objetivo:

✅ Organizar melhor o código
✅ Facilitar a manutenção do sistema
✅ Reutilizar código
✅ Tornar programas grandes mais fáceis de desenvolver
